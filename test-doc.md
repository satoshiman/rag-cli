# Test Document

This is a test document for the RAG CLI.

## Introduction

The RAG CLI is a local-first retrieval-augmented generation tool for personal documentation.

## Features

- Index markdown files
- Index text files
- Index PDF files
- Semantic search
- AI-powered answers

## Usage

Run `rag add ./docs` to index files.
Run `rag "your question"` to ask questions.
Run `rag status` to see index status.
